🚀 SPORTOUR (스포투어)
"스포츠와 여행의 완벽한 결합" > 전 세계 스포츠 팬들을 위한 맞춤형 스포츠 직관 및 로컬 여행 통합 플랫폼

📌 프로젝트 개요
SPORTOUR는 단순히 경기를 관람하는 것을 넘어, 스포츠 팬들이 최적의 동선으로 여행을 즐길 수 있도록 돕는 서비스입니다. 파편화된 경기 예매 정보와 여행 정보를 하나로 통합하여, 사용자 맞춤형 스포츠 패키지 및 커뮤니티 서비스를 제공합니다.

🎯 기획 배경
복잡한 예매 과정: 해외/국내 경기 티켓 예매의 높은 진입장벽

정보의 부재: 경기장 주변 숙박, 교통, 주변 관광지 정보 탐색의 어려움

팬덤 문화: 같은 팀을 응원하는 팬들과의 소통 및 동행 구인 니즈

✨ 주요 기능 (Key Features)
1. 스포츠 직관 패키지 예약
One-Stop 예약: 경기 티켓 + 인근 숙소 + 교통편을 한 번에 결합한 패키지 상품 제공

맞춤형 필터: 종목별, 리그별, 팀별 맞춤 검색 기능

2. 스마트 일정 플래너
동선 최적화: 경기 시간을 중심으로 주변 관광 명소와 맛집을 포함한 최적의 여행 루트 추천

인터랙티브 지도: API를 활용한 경기장 및 주요 거점 시각화

3. 팬 커뮤니티 및 동행 찾기
팀별 게시판: 응원하는 팀별 정보 공유 및 실시간 채팅

동행 서비스: 혼자 가기 어려운 직관 여행을 함께할 팀원 모집 기능

4. 실시간 여행지 정보 및 알림
외교부 데이터 활용: 입국 제한 국가에 대한 정보 제공 및 현지 날씨 정보 제공

💻 핵심 구현 사항 (Core Implementations)
1️⃣ 외부 API 연동 및 데이터 가공 (External API Integration)
다양한 출처의 데이터를 통합하여 사용자에게 일관된 정보를 제공하기 위해 API 통신 및 데이터 파싱 로직을 구축했습니다.

실시간 데이터 수집: 공공데이터 포털, 스포츠 관련 외부 API를 연동하여 경기 일정, 경기장 위치, 주변 관광지 정보를 실시간으로 호출합니다.

JSON 데이터 핸들링: 외부에서 받아온 이기종 데이터를 서비스 규격에 맞는 JSON 형식으로 직렬화/역직렬화하여 프론트엔드에 최적화된 API 응답을 제공했습니다.

데이터 정제: 불필요한 데이터를 필터링하고, 서비스에 필요한 핵심 정보만을 추출하여 응답 속도와 데이터 효율성을 높였습니다.

2️⃣ Spring Security를 이용한 인증 및 권한 관리
서비스의 보안성을 확보하고 사용자별 맞춤형 접근 제어를 위해 보안 로직을 설계했습니다.

사용자 인증(Authentication): Spring Security 체인을 활용한 로그인 기능을 구현하여 안전한 사용자 인증 프로세스를 구축했습니다.

권한 분리(Authorization): Role-based Access Control (RBAC)을 적용하여 일반 사용자와 관리자의 접근 권한을 엄격히 분리했습니다.

관리자 페이지: 경기 정보 등록, 패키지 상품 관리, 사용자 관리 등 민감한 기능은 관리자 권한을 가진 계정만 접근 가능하도록 설정했습니다.

보안 설정: 세션 관리 또는 JWT 도입 등을 통해 웹 서비스의 기본적인 보안을 구현했습니다.

3️⃣ Dialogflow 기반 지능형 챗봇 (Intelligent Chatbot)
사용자의 문의에 실시간으로 대응하기 위해 Google의 Dialogflow를 활용한 챗봇 서비스를 구축했습니다.

시나리오 기반 응답 설계: 예상되는 사용자 질문(경기 예매 방법, 여행 패키지 안내, 사이트 이용 방법 등)을 분석하여 **인텐트(Intent)**를 설정하고 최적의 답변 시나리오를 구축했습니다.

자연어 처리: 사용자의 질의 의도를 파악하여 미리 설정된 데이터베이스를 바탕으로 정확한 정보를 제공합니다.

고객 경험 향상: 24시간 자동 응답 시스템을 통해 관리자의 개입 없이도 반복되는 질문에 효율적으로 대응할 수 있도록 구현했습니다.

Frontend
